二维码绘制模块

依赖：jQuery，查看DEMO。

来源：http://larsjung.de/qrcode/

<script src="../jquery.qrcode-0.8.0.min.js"></script>

可选canvas、div、image

jQuery('#qrcode').qrcode({
    render    : "div",
    text    : "http://www.mogujie.com/",
    size    : 200, //200px
    ecLevel: 'L' //'L', 'M', 'Q' or 'H'，级别越高二维码越“复杂”
})

更多配置参见官网：http://larsjung.de/qrcode/

最佳实践

在IE6中需要强制设置字体、行高，避免干扰div的展示：

#qrcode div{
    font-size : 0;
    line-height : 0;
    overflow : hidden;
}
