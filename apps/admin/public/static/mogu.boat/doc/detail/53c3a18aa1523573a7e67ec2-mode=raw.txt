### 2 幻灯片播放插件swipe介绍（/jsmobile4/lib/swipe/swipe.js）
#### wap开发中有几个常用js，这次开发我用到是zepto,swipe。zepto简单点可以理解成精简版的jquery，所以这次介绍的重点为swipe。
#### 1 引用swipe
因为左右滑动这个交互在团购的模板中非常常见所以在配置模板的时候，把起和zepto一起当做base级js一起加入模板。

    $jsbase = array('/jsmobile4/lib/zepto/zepto.js','/jsmobile4/lib/swipe/swipe.js');
    _script2 ( $jsbase, 'base', true );

#### 2 swipe的使用
下面举一个简单的例子
    
    html代码
    <div id='slider' class='swipe'>
      <div class='swipe-wrap'>
        <div>1</div>
        <div>2</div>
        <div>3</div>
      </div>
    </div>


    js代码
    var mySwipe = new Swipe($('#slider')[0], {
        startSlide: 2,
        speed: 400,
        auto: 3000,
        continuous: true,
        stopPropagation: false,
        callback: function(index, elem) {},
    }); 

    添加了事件后我们还需要添加一些基础的css，以保证滑块能正常工作：
    .swipe {
      overflow: hidden;
      visibility: hidden;
      position: relative;
    }
    .swipe-wrap {
      overflow: hidden;
      position: relative;
    }
    .swipe-wrap  div {
      float:left;
      width:100%;
      position: relative;
    }

下面列举swipe配置一些参数：

**startSlide** (Integer) 滑动的索引值,即从*值开始滑动，默认值为0

**speed**      (Integer) 滑动的速度，默认值300毫秒

**auto**       (Integer) 自动滑动，单位为毫秒,默认值为0，等同于false。

**continuous** (Boolean) 是否循环滑动，默认值为true

**stopPropagation** (Boolean) 停止事件传播,默认值flase

**callback**        (Function) 回调函数


 
#### 3 使用swipe时的注意点

a) 初始化获取的swipe获取是原生dom不是jquery对象，所以要jquery对象后加[0]这样简单转一下

b) 需求通常会要求只有一张图的时候不能滑动。

c) 因为其实现原理，在幻灯片子元素只有两个时候并且设置continuous为ture时，swipe会多复制2个dom来时间左右无缝滑动，这样会照成页面性能的浪费。
解决上诉b+c遇到问题就是，一张幻灯片的时候不初始化swipe，两张的时候不设置continuous和auto。举例

    // 幻灯片存在，则初始化
    if( $("#head_slider").length > 0 ) initHeadSlider($("#head_slider"));
    function initHeadSlider(obj) {
        // 获取slideBox
        var slideBox = obj;

        // 如果ul个数大于1，初始化滑动组件,且只有两屏的时候无法回滚
        if(slideBox.find('.head_banner').length > 1) {
            var continuous = true;
            var auto = 2500;
            // 如果幻灯片为2时不自动滚动和循环滚动
            if (slideBox.find('.head_banner').length <= 2) {
                continuous = false;
                auto = false;
            }
            // 初始化swipe
            var mySwipe=new Swipe(slideBox[0],{
                auto:auto,
                continuous:continuous,
                callback: function(index, element) {
                    // 回调，使对应的小红点亮起
                    slideBox.find(".swipe_indicator li").removeClass("active").eq(index).addClass('active');
                }
            });
        }
    }
